# Mealgen
This will help to tell, what to cook based on the requirement, and show how to make a delicious dish


## Meal Generator Prototype

### Run locally
1) Create a venv (optional)
   python -m venv .venv
   source .venv/bin/activate

2) Install dependencies
   pip install -r requirements.txt

3) Start server
   uvicorn app.main:app --reload

Open:
http://127.0.0.1:8000
